# 🎧 HiFi Audio & Production Personal Wiki

## ⚙️ 활용법
1. **Zettelkasten / Obsidian 연동**:
   이 폴더 구조는 마크다운 기반의 지식 연결 도구(Obsidian 등)에 그대로 가져다 넣으면 폴더 정리와 문서간 링킹이 완벽하게 맵핑되어 즉시 개인 지식 기지로 사용 가능합니다.
2. **NotebookLM 다중 소스화**:
   개별 마크다운 파일들을 통째로 Google Drive에 올린 뒤 NotebookLM에 일괄 추가하여, 소스 레벨의 대화 정밀도를 극대화할 수 있습니다.
