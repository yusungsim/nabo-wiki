# SFTP (Secure File Transfer Protocol)

## 개요
네트워크 상에서 서버와 클라이언트 간 안전한 명령 전송과 데이터 채널을 체결하는 **SSH(Secure Shell) 프로토콜**의 암호화 터널 레이어 위에 FTP의 파일 제어 기능을 올려서 작동하는 보안성 극상의 암호화 파일 전송 및 스트리밍 프로토콜입니다.

## 음향 서버 활용 강점
외부 카페나 대중교통 이용 시 가상 VPN 터널링을 체결하지 못한 상태에서, 안전하지 않은 공용 공용 WiFi 망을 타고 집에 있는 맥북 홈서버 음원에 직접 연결할 때 패스워드나 음원 전송 패킷이 도청당하는 사고를 완벽 차단해 줍니다. 전송 헤더 데이터 오버헤드가 다소 있으나 기기의 연산 성능 향상으로 고음질 24비트 음원 스트리밍을 안정적으로 실시간 패킷 전송해 줍니다.
