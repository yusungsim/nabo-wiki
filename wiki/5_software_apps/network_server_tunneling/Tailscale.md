# 테일스케일 (Tailscale)

## 개요
복잡한 방화벽 포트포워딩 설정이나 도메인 구매 없이, 설치 한 번으로 본인의 모든 개인 기기(맥북 홈서버, 스마트폰, FiiO M21 DAP 등)들을 가상의 1대1 안전 암호화 가상 사설망(VPN)으로 즉시 묶어주는 혁신적인 메쉬 VPN 네트워크 도구입니다.

## 동작 원리 및 음향 홈서버 활용
* **WireGuard 프로토콜 기반**: 현존 최고속/초경량 암호화 터널링 규격인 WireGuard를 내장하여 모바일 배터리 소모와 레이턴시를 최소화합니다.
* **공유기 포트 개방의 위험 해소**: 보통 집 안 맥북 홈서버에 저장된 음원을 밖에서 들으려면 통신사 공유기의 특정 포트를 인터넷 전체로 개방(Port Forwarding)해야 하여 해킹 위험에 항시 노출됩니다. 
* **안전한 가상 인트라넷**: 맥북 서버와 외부 DAP에 `Tailscale`을 설치하고 로그인하면, 통신사가 강제 부여한 공인 IP와 상관없이 기기 간 고유 가상 IP(예: `100.x.x.x`)가 배정됩니다. 외부 야외망(LTE/5G)에서 이 고유 가상 IP로 접속 요청을 하면 방화벽을 우회하여 마치 집 안 내방 공유기 WiFi에 물리 연결된 것처럼 안전하게 홈서버 음원을 실시간으로 빨라 당겨 스트리밍 감상할 수 있습니다.
