# 💻 네트워크 서버 및 터널링 (Servers & VPN)

맥북을 고음질 홈 오디오 서버로 전환하여 외부망 언제 어디서나 안전하게 무손실 라이브러리를 스트리밍하게 해주는 인프라 도구 인덱스입니다.

---

## 🗂️ 세부 개념 목록
* [나비드롬 (Navidrome)](Navidrome.md) - 메모리/CPU 리소스 점유가 극도로 적고 Subsonic 표준 스펙을 제공하여 외부 UAPP/Neutron 플레이어와 최적 매칭되는 오픈소스 무손실 음원 서버.
* [Plex 미디어 서버](Plex.md) - 개인 컴퓨터 및 하드의 영화, 음악을 일괄 통합 라이브러리화하며 음악 감상 시 전용 Plexamp 플레이어와 실시간 결합되는 명기 미디어 허브.
* [테일스케일 (Tailscale 메쉬 VPN)](Tailscale.md) - WireGuard 프로토콜을 사용해 공유기 포트포워딩 개방 위험 없이 홈서버 맥북과 내 폰을 안전한 1:1 암호 가상 터널로 연결해주는 가상 사설 VPN.
* [클라우드플레어 터널](Cloudflare_Tunnels.md) - 외부 접속용 클라이언트 VPN 앱 설치 없이, 안전한 아웃바운드 터널 보안을 통해 본인의 개인 웹서버 주소를 안전하게 공용 도메인 주소로 노출해주는 도구.
