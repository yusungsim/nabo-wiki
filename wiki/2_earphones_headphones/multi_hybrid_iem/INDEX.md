# 🎧 다중 / 하이브리드 인이어 (Multi / Hybrid IEM)

각기 다른 드라이버의 강점을 크로스오버 네트워크 회로로 정밀 조합하여 다이내믹 레인지와 해상도를 극대화한 이어폰 목록입니다.

---

## 🗂️ 세부 개념 목록
* [TRN 캐러벨](TRN_Caravel.md) - DD의 저역, BA의 중역, PZT/EST 복합 트위터의 고역 조합으로 시각적인 다중 텍스처를 뿜어내는 가성비 하이브리드.
* [키위이어스 오케스트라 2 / 라이트](Kiwi_Ears_Orchestra.md) - 8개의 BA만을 유기적으로 엮어 왜곡이 거의 없는 플랫하고 깔끔한 프로 모니터링 밸런스를 구현한 풀 BA 명기.
* [7Hz 소너스](7Hz_Sonus.md) - DD의 깊은 저음 울림과 BA의 또렷한 목소리 분리를 깔끔하게 매치하여 정돈된 하만 타겟 사운드를 선사하는 리시버.
* [수월우 메이](Moondrop_May.md) - 사파이어 DD와 소형 평판 트위터 구성에 하드웨어 DSP 칩이 박힌 C타입 DAC 케이블을 제공하여 완전 디지털 보정이 가능한 이어폰.
* [탕주 부두 (Fudu Verse 1)](Tangzu_Fudu.md) - 1DD + 2BA 구성의 차분하고 부드러운 튜닝. 패키지 내 디비누스 벨벳 팁 일반형이 번들 포함되어 저음 차음력을 고도화함.
* [심고트 EG280](Simgot_EG280.md) - 심고트 고유의 정교한 어쿠스틱 제어로 무대 공간과 악기 정위 배치가 뛰어난 모니터링 대응 리시버.
* [케파인 아르나르](Kefine_Arnar.md) - 가성비 라인업에서 각 음역대의 조화로움과 단단한 타격을 골고루 확보한 올라운더형 멀티 드라이버 하이브리드.
